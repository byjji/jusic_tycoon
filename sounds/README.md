# 🔊 효과음 파일 넣는 곳

이 폴더(`sounds/`)에 아래 **파일명**으로 음원을 넣으면 게임이 자동으로 사용합니다.

- 확장자는 **mp3 / wav / ogg** 중 아무거나 가능 (같은 이름이 여러 개면 mp3 → wav → ogg 순으로 사용)
- 파일이 **없으면 Web Audio 합성음으로 자동 대체**되므로, 일부만 넣어도 됩니다.
- 파일을 추가/교체한 후 브라우저 새로고침만 하면 적용됩니다.

| 파일명 | 용도 | 권장 길이 / 형태 |
|---|---|---|
| `coaster_loop` | 롤러코스터 주행 굉음 | 2~10초, **루프용** (끊김 없이 이어지는 소리) |
| `wind_loop` | 바람 소리 | 2~10초, **루프용** |
| `lift_clack` | 리프트 체인 "딸깍" 1회 | 0.1~0.3초 (오르막에서 자동으로 연속 재생됨) |
| `siren_loop` | 급락 경고 사이렌 | 2~6초, **루프용** |
| `scream` | 비명 | 1~2초 |
| `firework` | 폭죽 터지는 소리 | 0.5~1.5초 |
| `launch` | 로켓 발사음 (우주 엔딩) | 3~6초 |
| `crash` | 충돌·폭발음 (추락 엔딩) | 1~3초 |
| `brake` | 브레이크 마찰음 (정지 엔딩) | 1~2초 |

예시: `sounds/scream.mp3`, `sounds/siren_loop.wav`, `sounds/firework.ogg`

참고: 주행 굉음과 바람은 속도에 따라 음량과 재생 속도(피치)가 자동으로 조절됩니다.
